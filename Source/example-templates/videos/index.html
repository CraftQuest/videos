Setup Instructions

Some section of this example application require specific setup.

Video Details

No setup required

Video Field

  1. Create a new Videos field and call it video.
  2. Create a new global and call it Videos Demo.
  3. Add the video field to Videos Demo global.

Video Field with Matrix

  1. Create a new Matrix field and call it videos.
  2. Add a new block type to this Matrix field and call it video.
  3. Add a new field to this block type and call it video (yep, again).
  4. Create a new global and call it Videos Demo.
  5. Add videos Matrix field to Videos Demo global.
